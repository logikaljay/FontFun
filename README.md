Font Fun
=======

lunchtime fun with font awesome

## build
`make`


## deploy
```
scp ./ff root@iphone.local:/var/root/ff
scp ./FontAwesome.ttf root@iphone.local:/var/root/FontAwesome.ttf
```


## run
```
ssh root@iphone.local
./ff
```

## test
```
scp root@iphone.local:/var/root/test.png ./test.png
open ./test.png
```

enjoy your heart!
